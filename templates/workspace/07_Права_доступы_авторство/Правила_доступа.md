# Правила доступа

- `public` - открытые утвержденные материалы.
- `resident` - материалы для активных резидентов в согласованном контуре.
- `private` - материалы владельца рабочего места.
- `temporary` - сырье для локальной обработки.
- `pending_rights` - права или режим не определены; материал не передается.

Изменение режима доступа не меняет авторство автоматически.

Codex не публикует, не отправляет, не загружает и не меняет права доступа без
явного подтверждения действия и назначения.

Отключение от GitHub закрывает будущий доступ к обновлениям, но не удаляет ранее
скачанную локальную копию. Правила оферты и авторства продолжают действовать.
