<p>📝 Документы DocKA: Календарь инфоповодов (DocKA3.2.3.1),<br>
<a class="" href="https://alliance.beinopen.ru/post/200/">Календарь стимулирования продаж (календарный план маркетолога) (DocKA3.2.3.3)</a>, Пример бюджетирования по рекламным каналам (DocKA3.2.3.2):</p>
<p>Собираем стандарт фреймов и документов, по которым работает отрасль. В эфире разберем аспекты работы, необходимые для создания маркетингового плана, обсудим, как с этими документом работать, как его анализировать.</p>
<p>Будем обсуждать следующие аспекты создания маркетингового плана:</p>
<ol>
<li>Резюме (краткое описание маркетинговой стратегии).</li>
<li>Анализ ситуации (анализ рынка, анализ конкурентов, анализ ЦА, SWOT-анализ)</li>
<li>Цели (узнаваемость, продажи, лояльность аудитории, трафик)</li>
<li>Стратегия позиционирования и УТП (📄 <a class="" href="https://alliance.beinopen.ru/post/2941/#1-Kratkii-freimvork-BRA">Краткий фреймворк разработки маркетинговой стратегии. BRAND / MARKETING / ACTIVATION (DocKA3.5)</a>)</li>
<li>Тактика и календарный план (DocKA3.2.3.1)</li>
<li>Бюджет (DocKA3.2.3.2)</li>
<li>Метрики и аналитика</li>
</ol>
