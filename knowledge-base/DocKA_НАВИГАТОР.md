# DocKA: рабочие документы резидентского контура

[DocKA](https://alliance.beinopen.ru/docs/2046/) дополняет статьи Свода знаний
рабочими документами: стратегиями, таблицами, договорами, календарями,
техническими заданиями, аналитическими формами и чек-листами.

[Открыть карту связей KA -> DocKA](./DocKA_КАРТА_ДОКУМЕНТОВ.md)

```text
KA объясняет документ и место решения в системе
-> DocKA дает рабочую форму
-> резидент заполняет ее в приватном рабочем месте
-> результат становится моделью компании, TR или согласованным кейсом
```

## Ладошка Сачек

- KA1 — стратегия компании;
- KA2 — бренд-стратегия;
- KA3 — маркетинговая стратегия;
- KA4 — ассортиментная стратегия;
- KA5 — коммерческая стратегия.

Эти пять верхнеуровневых документов должны удерживать единый каркас компании и
не противоречить друг другу.

Сами шаблоны DocKA не включаются в открытый GitHub. Они доступны активным
резидентам по условиям оферты. Нужный документ выбирается через KA6/KA,
проверяется на платформе и выборочно скачивается из управляемого Yandex Disk.
Полный архив в рабочее место не устанавливается. Актуальный статус доступа,
версия и авторство проверяются перед получением файла.
