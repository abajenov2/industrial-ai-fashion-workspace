# CJM рабочего места резидента

CJM описывает пошаговый путь человека или организации через систему Альянса.
Он не заменяет KA, роль или протокол.

```text
KA = содержание знания
CJM = последовательность действий и контрольных точек
RL = действующая роль
TR = фактический результат прохождения пути
```

Каждый утвержденный CJM должен иметь:

- стабильный ID и версию;
- владельца стандарта;
- входное и выходное состояние;
- шаги, роли, источники и контрольные точки;
- правила доступа;
- связанные KA, DocKA, RLTR, TRxTR и A1.x;
- критерий завершения и следующий маршрут.

Машинный индекс: [`machine/cjm-index.yaml`](./machine/cjm-index.yaml).
