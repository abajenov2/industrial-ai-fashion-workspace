# Первый запуск рабочего места

## Вход

Владелец хочет подключить Codex и библиотеку Альянса, но рабочее место еще не
создано или не заполнено.

## Шаги

| Шаг | Действие | Результат |
|---|---|---|
| 1 | установить пакет в новую пустую папку | единое рабочее место создано |
| 2 | открыть `AGENTS.md` и политику | Codex знает границы работы |
| 3 | заполнить владельца и профиль Альянса | определена субъектность |
| 4 | отделить организационные контексты от ролей | нет смешения марки, фабрики и должностей |
| 5 | выбрать 1-3 действующие роли RL | появилась рабочая конфигурация |
| 6 | назвать задачу на 24-48 часов | установка связана с практикой |
| 7 | пройти один маршрут через KA6 | найдено действие с датой |
| 8 | записать журнал установки | собрана обратная связь для улучшения пакета |

## Выход

- заполнен минимальный паспорт;
- указаны роли и ближайшая задача;
- проверены доступы;
- создано действие с владельцем и датой;
- журнал не отправлен наружу без отдельного подтверждения.
