# Safety Rules

Use these rules whenever a task touches access, publication, uploading, messaging, transcripts, personal data, or third-party sharing.

## Access Levels

- `public`: may become public after review and user approval.
- `resident`: only for registered Alliance residents / "only for ours".
- `private`: personal, contractual, sensitive, unapproved, or not intended for sharing.
- `temporary`: raw transcripts, exports, working copies, unprocessed files.

If access is unclear, treat the material as `private` until the resident decides.

## Actions Requiring Explicit Approval

Ask for explicit confirmation before:

- publishing to the Alliance platform;
- posting comments;
- sending messages to people;
- uploading files to Google Drive or another service;
- changing access permissions;
- sharing resident data, raw transcripts, private notes, exports, contact details, or internal documents;
- deleting or moving important files outside the workspace.

Approval must name the action and destination.

## Prohibited in Open Skill Files

Do not store:

- passwords;
- API keys;
- cookies;
- payment data;
- private Zoom / meeting access codes;
- raw personal data exports;
- unapproved resident profiles, private correspondence, or raw transcripts.

## Transcript Handling

Raw transcripts are temporary and private by default.

Workflow:

```text
raw transcript
-> temporary file
-> processed artifact
-> access classification
-> review
-> publication or archive decision
```

Processed artifacts may become:

- trajectory (`TR`);
- trajectory intersection case (`TRxTR`);
- interview;
- meeting protocol;
- Knowledge Base draft;
- task list;
- site update.

`TRxTR` is not a privacy level and not `A1.x`. It is a case where several trajectories intersect. A single `TRxTR` may generate private, resident, or public derivatives. `A1.x` should only be proposed after several similar `TRxTR` cases show a repeatable modular assembly that can be described through stakeholder needs, a requirements matrix, target indicators, resource constraints, typical modules, validation, and repeatability.

Approved `TRxTR` artifacts usually publish to the Alliance room `AI оптимизация`. This route does not override access rules: sensitive data, raw transcripts, commercial details, private expert conclusions, and unapproved personal data must stay private, be anonymized, or be published only in the approved closed mode.

## Website Rule

The Alliance platform is the source of truth for public and resident-facing standards. Local files are working copies until a change is approved and published.
