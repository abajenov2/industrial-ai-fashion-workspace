# Industrial AI Fashion Workspace

Открытый стандарт рабочих мест резидентов Альянса x Beinopen.

До первого внешнего release встроенный Свод считается кандидатом: его нужно
полностью обновить из эталона роли Архитектора и сверить по `KA1-KA6`, авторам,
доступам, DocKA-связям и RLTR. Целевая дата первой базовой версии `1.0.0` -
2026-07-22; после нее учебник продолжает обновляться по версиям.

Репозиторий содержит:

- полную открытую версию Свода знаний;
- карточки-ссылки на материалы резидентского контура;
- индексы авторов, экспертов, кейсов, TRxTR, событий и профилей;
- KA6-навигацию: как определить положение, найти маршрут и выбрать следующий шаг;
- RLTR-навигацию от вопроса к действию и подходящим партнерам;
- шаблоны ролей марки, эксперта, фабрики и архитектора;
- установщик и инструмент обновления общей библиотеки;
- Codex skill для безопасной работы в контуре Альянса.

Тексты, структура и авторство статей зеркалируются полностью. Изображения, видео
и внешние документы остаются ссылками на исходные публикации и хранилища.

## Установка

```bash
bash install.sh brand /absolute/path/to/brand-workspace
bash install.sh expert /absolute/path/to/expert-workspace
bash install.sh factory /absolute/path/to/factory-workspace
bash install.sh architect /absolute/path/to/architect-workspace
```

После установки откройте созданную папку в Codex. Codex прочитает `AGENTS.md`,
правила доступа, KA6- и RLTR-навигаторы, затем поможет заполнить паспорт и провести
первую реальную задачу через Свод знаний.

## Обновление Свода

Сначала обновите локальный clone репозитория, затем запустите:

```bash
bash update-library.sh /absolute/path/to/resident-workspace
```

Обновляется только согласованный открытый корпус. Приватные документы резидента
и выборочно полученные DocKA-файлы не затрагиваются.

Новая версия сначала собирается и проверяется в локальном рабочем месте
Архитектора. Только затем создается GitHub release, который может получить
рабочее место резидента.

DocKA не поставляется полным архивом через GitHub. Нужный документ выбирается по
KA6/KA, открывается на платформе и скачивается из управляемого Yandex-каталога
после проверки доступа.

## Граница данных

В открытом репозитории нет cookies, токенов, контактных баз, приватных переписок,
чужих сырых транскриптов и коммерческих данных резидентов. Актуальные контакты
открываются через профили на платформе Альянса.
